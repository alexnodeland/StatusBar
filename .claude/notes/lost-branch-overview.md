## Re-implementation Strategy

The MEMORY.md in `.claude/projects/` contains architectural decisions and gotchas learned during original implementation. Key references:

- Bundle ID: `com.alexnodeland.statusbar` (not `com.statusbar.app`)
- Widget target only has `Widget/` + `Sources/SharedStatusData.swift`
- Test target excludes: `StatusBarApp.swift`, `HotkeyManager.swift`, `AppleScriptBridge.swift`, `ShortcutsIntents.swift`
- Widget needs custom `Widget/Info.plist` (INFOPLIST_KEY_ doesn't generate nested NSExtension keys)
- Root-level resources need `buildPhase: resources` in sources section for xcodegen
- Test target `dependencies: []` (no host app)
- Swift 5 language mode (not Swift 6 strict concurrency)
